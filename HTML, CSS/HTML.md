## HTML(HyperText Markup Language)?

### 웹 페이지 표시를 위해 개발된 지배적인 마크업 언어다.

- HyperText: 문서와 문서가 링크로 연결됨
- Markup: 태그로 이루어짐
- Language: 웹 페이지가 이해할 수 있는 언어

### 형식

#### <태그명 속성명1='속성값1' 속성명2='속성값2'> 컨텐츠 </태그명> 

### 문서의 구조
```html
<html>
  <head>
    문서를 정의하는 데이터가 위치
  </head>
  <body>
    문서에 표시되는 컨텐츠가 위치
  </body>
</html>
```
